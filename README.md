# 🌍 AI Multilingual Translator with Custom Audio Synthesis

An advanced AI-powered translation platform that converts multiple global languages into English with intelligent detection and customizable audio output. Designed with a focus on user experience, accessibility, and modern UI.

---

## 🚀 Project Overview

This project aims to bridge communication gaps by providing seamless language translation combined with personalized speech synthesis. It not only translates text but also enhances interaction through customizable voice output and dynamic user interface themes.

---

## ✨ Key Features

### 🧠 Intelligent Language Detection

* Automatically detects the source language using AI
* Eliminates the need for manual language selection
* Provides fast and seamless translation to English

---

### 🔊 Advanced Audio Customization

* 🎙️ Multiple voice profiles for different contexts
* ⚡ Adjustable speech speed for better clarity
* 🎵 Pitch & bass control for natural or stylized output
* 🔉 Enhanced accessibility for diverse user needs

---

### 🎨 Personalization & UI/UX

* 🌈 Dynamic themes for better visual experience
* 💡 Clean and intuitive interface
* ⚡ Smooth navigation from input → translation → audio output

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, TypeScript, Vite
* **AI/NLP:** Language detection & translation models
* **Speech Processing:** Text-to-Speech (TTS) with custom controls
* **Runtime Tools:** Node.js, npm

---

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/ShreejaMandaloju/AI-Translator-Enhanced.git

# Navigate to project folder
cd AI-Translator-Enhanced

# Install dependencies
npm install --legacy-peer-deps

# Run the project
npm run dev
```
